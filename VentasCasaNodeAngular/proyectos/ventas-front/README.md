# Ventas Front (plantilla)

Este es un esqueleto para el frontend (Angular) del proyecto "Ventas Casa".

Pasos rápidos:
1. En `proyectos/ventas-front` ejecutar `npm install`.
2. Si usas Angular CLI instalado localmente, ejecutar `npm start` o `ng serve --port 4300`.
3. Configura `environment` para apuntar a `http://localhost:3500` (backend).
